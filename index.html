HoloHUD

A hud mod for Payday 2

GitHub page Download(3.3.1) Steam group

HoloHud is a payday 2 mod which focuses on how the hud looks and adds tons features!

Its also a very customizable hud with many options.

How to install :

  1. Download HoloHud : Download
  2. Download BLT or BLT:CSE if you're using HoxHud
  3. Drop the two files in the zip file to your payday 2 directory
  4. You're done!
