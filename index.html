HoloHud is a payday 2 mod which focuses on how the hud looks and more stuff. Its also a very customizable hud with many options. This mod requires BLT - http://paydaymods.com/download/ if you use HoxHud use BLT:CSE : https://github.com/Olipro/Payday-2-BLT_Club-Sandwich-Edition/releases
