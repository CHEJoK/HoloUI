Reborn!
     HoloHUD is Reborn!, Now all of the assets of HoloHUD are inside the mod itself making it much easier to install and update the mod
     Many new features and now uses BeardLib!
What is HoloHUD?
     HoloHUD is a mod that focuses mostly on the look of the hud and menu and contains tons of customization options

How to install? 
     First you need to get BLT and BeardLib, After that move the folder inside the zip file to mods folder.
